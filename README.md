# python-data-analytics-projects
# Python Data Analytics Projects

A curated collection of Python data analytics projects completed using real-world datasets.  
These notebooks demonstrate skills in **data cleaning**, **EDA**, **visualization**,  
**statistical analysis**, **feature engineering**, and **business insights** using  
Pandas, NumPy, Matplotlib, Seaborn, and Jupyter Notebooks.

This repository is designed to showcase end-to-end analytical thinking for  
**data analyst, business intelligence, and data science internship roles**.

---

## **Projects Included**

### 1. **ASOS Revenue Analysis**
- Cleaned and analyzed e-commerce revenue data to identify sales trends.
- Conducted monthly revenue breakdowns and customer behavior insights.
- Tools: Python, Pandas, Matplotlib.

---

### 2. **College Football Performance Analysis**
- Explored offensive and defensive performance metrics.
- Identified predictors of team wins using correlation and trend visualizations.
- Tools: Python, Pandas, Seaborn.

---

### 3. **NYC Schools Analysis**
- Analyzed DOE school performance metrics across boroughs.
- Uncovered graduation rates, demographic patterns, and funding relationships.
- Tools: Python, Pandas, Seaborn, Jupyter.

---

### 4. **Olympics Dataset Analysis**
- Investigated medal trends, athlete demographics, and country-level performance.
- Built visual dashboards for multi-year comparisons.
- Tools: Python, Matplotlib, Pandas.

---

### 5. **Planetary Weather Data Analysis**
- Compared weather patterns across multiple planetary bodies (NASA dataset).
- Identified surface temperature variations and atmospheric pressure cycles.
- Tools: Pandas, NumPy, Line Charts.

---

### 6. **Website Performance ‚Äì Grammys Edition**
- Analyzed web traffic and engagement metrics around Grammy events.
- Highlighted peak interaction hours and performance bottlenecks.
- Tools: Python, Pandas, Matplotlib.

---

### 7. **Doctor Who Quiz Data Analysis**
- Cleaned a messy TV quiz dataset and extracted patterns in answers.
- Performed insight extraction using groupby and statistical summaries.
- Tools: Pandas, Data Cleaning techniques.
